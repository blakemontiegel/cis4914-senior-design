Youth soccer club video management web application

## User quick start

[Open Sideline](https://jettnguyen.github.io/Sideline/)

## Developer quick start (backend + frontend)

### 1. Backend setup

Install and run the backend:

```bash
cd backend
npm install
npm run dev
```

### 2. Frontend setup

Install and run the frontend:

```bash
cd frontend
npm install
npm start
```
