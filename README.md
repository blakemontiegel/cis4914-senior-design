## What is Sideline?

Sideline is a web app built for youth soccer teams. It gives coaches and parents one place to organize match footage, tag key moments, manage team membership, and share game context without juggling multiple tools or group chats.

---

## Get Started

### Open in a browser

**[jettnguyen.github.io/Sideline](https://jettnguyen.github.io/Sideline/)**

### Install on your phone (recommended)

Sideline is a Progressive Web App. Install it like a native app for the best experience.

#### iOS (Safari)

1. Open Sideline in Safari
2. Tap the **Share** button
3. Tap **Add to Home Screen**
4. Confirm **Open as Web App** is enabled
5. Tap **Add**

#### Android (Chrome)

1. Open Sideline in Chrome
2. Tap the browser menu (three dots)
3. Tap **Add to home screen**
4. Confirm installation

## Features

| Feature | Description |
| --- | --- |
| Team management | Create teams, invite members, or approve join requests |
| Roles | Assign coach, parent, or player roles per team |
| Match log | Add matches and attach video clips |
| Video tagging | Mark key moments while watching clips |
| Profiles | Browse clips by match, player, or profile |
| Mobile-first | Installable PWA that works great on any device |

---

## Tech Stack

- **Frontend**: React, PWA
- **Backend**: Node.js + Express
- **Database**: MongoDB + Mongoose
- **Media storage**: AWS S3
- **Upload pipeline**: Uppy

---

## Developer Quick Start

### 1. Backend

```bash
cd backend
npm install
npm run dev
```

### 2. Frontend

```bash
cd frontend
npm install
npm start
```
